
###############
Global Workflow 
###############

**Global-workflow** is the end-to-end workflow designed to run global configurations of medium range weather forecasting for the UFS weather model. It supports both development and operational implementations. In its current format it supports the Global Forecast System (GFS) and the Gobal Ensemble Forecast System (GEFS) configurations

======
Status
======

* State of develop (HEAD) branch: early GFSv17+ development

* State of operations branch: GFS v16.3.5 `tag: [gfs.v16.3.5] <https://github.com/NOAA-EMC/global-workflow/releases/tag/gfs.v16.3.5>`_
